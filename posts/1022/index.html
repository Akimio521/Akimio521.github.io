免费的云数据库推荐

发表于2023-06-05|更新于2023-10-01|备忘录
|字数总计:1.1k|阅读时长:3分钟|阅读量:|评论数:

前言

一个星期前收到了Bit.io的邮件，说是要跑路了，6.29之后数据库进入只读模式，7.29前保持可访问，如果大家有用Bit.io数据库的准备要在23.6.30前迁移数据库内容。Bit.io算是我最喜欢的一款免费云数据库了吧，Bit.io是托管在GCP云的Postgres云数据库，美国VPS连接速度不错，国内也可以连接。免费账号可以创建3个数据库，3G的空间和每月10亿次查询。Bit.io界面简洁明了美观，除了没有中文。在之前两次搭建Alist的教程中，我也是首推了Bit.io（有兴趣的可以看看：利用Replit搭建Alist、在Render上部署Alist）。可能是薅羊毛的人太多了吧，Bit.io跑路了，为这款这么好的免费云数据库的离开感到遗憾和惋惜。借此机会，我着了网上几款比较好的免费云数据库供大家学习，请不要用于商用，也不要滥用免费资源

MySQL

PlanetScale

PlanetScale官网地址，可以使用Github账号注册，不支持大陆IP，区域一定要选择 AWS / N. Virginia (us-east-1)

数据库基本信息 示例
Host us-east.connect.psdb.cloud
Port 3306

db4free

db4free.net提供最新版本、甚至是开发者版本的MySQL 服务器测试服务，每当MySQL有新版本推出，db4free.net 的 MySQL 服务器通常将会在同一天或几天内更新至最新版本。

db4free服务器在国外，可能比较慢但是是稳定的。并且无需注册账号就能申请数据库，数据库有200M的存储空间，官方会不定期清理超过200M的部分并且不做任何通知。如果你的数据不是很大，用db4free是一个不错的选择。无需注册账号即是一个优点也是一个缺点吧，没办法管理自己所有的数据库，想删库也不是很方便，容易造成资源浪费。提供phpMyAdmin在线数据管理

数据库基本信息 示例
Host db4free.net
Port 3306

Sqlpub

Sqlpub服务器在国内，对于国内访问较多的可以看看。无需注册账号，最大36000次/小时的请求、同时10个连接和500M存储空间，并且提供了Web可视化界面管理数据库

数据库基本信息 示例
Host my.sqlpub.com
Port 3306

Remotemysql

Remotemysql提供了永久100M的空间，访问比db4free快、提供 phpMyAdmin 在线数据管理、没有查询限制并且不限使用带宽，但是广告很多，综合来看是一个一般的免费在线 MySQL 数据库，不能称之为“服务”因为压根不放心存储数据。

FreeDB

FreeDB免费提供50MB数据库，最大查询数为800/h，每小时最大更新数为800，最大连接数为800。不能使用临时邮箱注册，每个人只允许拥有一个帐户，提供phpMyAdmin数据库管理面板，境内访问较慢。

数据库基本信息 示例
Host sql.freedb.tech
Port 3306

PostgreSQL

SupaBase

Supabase 是一个开源的Firebase替代品（Github仓库），Postgres数据库托管只是其中一部分，如果你在用SupaBase的其他产品也可以看看他们家的云数据库。数据库区域一定要选择 N. Virginia (us-east-1)

数据库基本信息 示例
Host db.xxx.supabase.co
Port 5432

Elephantsql

elephantsql 提供了 20 MB 额度的在线 PostgresSQL 数据库，由一家印度公司支持，有香港节点比较快

数据库基本信息 示例
Host john.db.elephantsql.com
Port 5432

MongoDB

MongoDB 是一个基于分布式文件存储的数据库。由 C++ 语言编写。旨在为 WEB 应用提供可扩展的高性能数据存储解决方案

我的个人博客的评论就是使用了MongoDB数据库作为存储
免费是数据库用的是共享虚拟主机，有500m的空间
创建免费MongoDB数据库, 区域选择 AWS / N. Virginia (us-east-1) 在 Clusters 页面点击 CONNECT, 记得要设置允许所有 IP 地址的连接）

数据库基本信息 示例
Host mongodb+srv://cluster0.xxxx.mongodb.net
Port 27017

尾声

以上就是我汇总的免费云数据库，如果你还有其他推荐的数据库也欢迎留言
