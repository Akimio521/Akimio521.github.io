使用BetterNCM美化网易云

|字数总计:1.4k|阅读时长:4分钟|阅读量:|评论数:

前言

最近网易云更新了Windows的3.0测试版本，样式也更加美观简洁，这里附上尝鲜版本：阿里云盘



样式比之前的版本好看多了，之前我是搭配BetterNCM进行美化，这篇文章主要是为了记录我的网易云美化方式
下面是我美化好的样式


插播一个广告

  • 欢迎使用我的音乐站，支持网易云音乐登入和灰色音乐解锁
  • 欢迎使用我的音乐解锁站，支持网易云、QQ、酷狗、酷我加密音乐格式解锁

BetterNCM

介绍

BetterNCM 是一个由一群热爱网易云音乐的云村村友开发的 PC 版网易云音乐扩展工具，可以提供非常丰富的自定义功能扩展增强能力

安装教程

这里提供了3种BetterNCM的安装方式，大家可以根据自己的实际情况选择安装方式，安装教程来源官方文档

仅支持64位系统，32位系统请手动安装
可以选择从官网下载或者从Github中下载

  • 官网
  • Github
    下载完成后运行exe可执行文件，点击安装，等待安装完成即可。如果一直卡在正在下载：betterncm.dll，请尝试再次点击安装按钮
    安装完成后打开网易云，右上角处会新增一个齿轮图标，就在网易云设置图标右边，看到齿轮图标就代表安装成功了

先下载文件

  • 稳定版
  • 测试版
  1. 下载完成后将文件由BetterNCMII.dll重命名为msimg32.dll
  2. 将msimg32.dll移动到网易云的安装目录下（默认安装目录为为C:\Program Files (x86)\Netease\CloudMusic）。如果找不到安装目录，可以在桌面右键网易云图标，点击打开文件所在的位置
  3. 复制完成后，重新打开网易云，右上角处会新增一个齿轮图标，就在网易云设置图标右边，看到齿轮图标就代表安装成功了

此方法近适用于UWP版本网易云

并不是很推荐使用UWP版本的网易云，有条件的用户请更换

这里提供UWP的整合包，可以直接安装带有BetterNCM的UWP网易云，以 MSIX 方式安装的网易云

UWP版会对文件进行保护，可以在PE下进行安装

  1. 使用快捷键WIndows键+r输入system control然后回车,找到高级系统设置->高级->环境变量

  2. 分别在Dell的用户变量和系统变量中找到BETTERNCM_PROFILE（没有的话自己新建变量），想要修改某个用户的网易云就在Dell的用户变量进行操作；想在该电脑上所有用户的网易云都进行修改就在系统变量上进行操作。

  3. 双击任意一个（或者新建），点击浏览目录选择或者创建一个文件夹，（文件夹路径不能有中文）这里我设置为c:/better_ncm即C盘中的better_ncm文件，修改完成后点击确定
  4. 重启电脑。重启后打开网易云
  5. 到C盘中的better_ncm文件夹中查看有没有文件生成，看到后plugins和plugins_runtime这两个文件夹就代表安装成功了

BetterNCM插件推荐

插件可以从官方的插件市场中下载，这里就列出我用的插件

美化插件

  • Material You：美化网易云界面，样式参考发现页，有多种主题选择，可以跟随系统明暗主题，可以自定义主题色
  • RefinedNowPlaying：重新设计网易云的播放页和歌词，界面更见美观，可以对封面、背景、歌词字体等项目进行调整
  • MikuPlugin：可以自定义按钮，让不想看到的按钮都见鬼去吧

拓展插件

  • Inflink：连接各种音乐协议，支持SMTC（Windows自带）、Discord RPC（Discord状态显示）
  • 纯净分享链接：插件如其名，去除分享链接中的用户信息
  • 任务栏歌词：类似于安卓上的状态栏歌词
  • FluentProgressBar：网易云原本的音乐进度条一卡一顿的，可以用这个改善，让进度条的变化更加丝滑刘畅

其他

  • 可以在官方文档了解插件之前也没有相互冲突的（特别是美化类插件）
  • 安装插件时会自动安装一些依赖插件或者依赖库，请不要随意卸载
  • 有些美化插件会使用流体计算，如果使用核显渲染卡顿且电脑有独立显卡，可以进入Windows设置->系统->屏幕->显示卡，将网易云音乐设置为高性能（独显渲染）

尾声

网易云公测版的界面样式更加好看，BetterNCM可能会带来网易云使用的卡顿、CPU占用增加、耗电增加等问题，原则上是能不用最好，目前BetterNCM并未对新版网易云适配，或许BetterNCM消失的那天就说明官方网易云已经做的足够好了吧或者网易云倒闭了

鸣谢

感谢为BetterNCM项目做出贡献的所有成员，正是因为你们的付出才让我们用上更好的云音乐
